## Reading these numbers honestly

- Keyword R@5 88.0% independently corroborates the earlier 90.0% (36/40) measured on a different
  gold set (reports/recall-experiments/2026-06-11-ranking-phase2.md). Two independent gold sets,
  same corpus family, consistent answer.
- Hybrid trails keyword by 14 points at corpus scale, the opposite of the public LongMemEval-S
  result. Root cause (investigated 2026-07-07, pre-registered): the metadata tiebreaker is applied
  only in keyword mode; hybrid's finalize path never receives the query tokens. The fix is known
  and deferred to avoid overlapping a running measurement window. A semantic-leg score floor was
  tested analytically and has zero effect; the drag is entirely the tiebreaker gap.
- Keyword R@1 24.0% vs the June set's 70.0% is gold-set composition, not an instrument defect:
  the June set was decision queries only, the exact class the tiebreaker discriminates; four of
  the six categories here get near-zero tiebreaker benefit. Corpus growth (2,968 to 5,502
  sessions) is a secondary contributor within the decision category.
- Single-operator corpus, n=50: treat point estimates as indicative. The value of this benchmark
  is the trend across reruns on the same locked set, not any single number.
